# Daily Reflection System

This project contains two parts:

## Part A
A deterministic decision tree stored as a TSV file.

## Part B
A Python-based AI agent that interacts with users and generates reflection.

## Features
- Structured decision-making
- Predefined options (reduces errors)
- Input validation (guardrails)
- Reflection based on:
  - Locus (Internal/External)
  - Orientation (Contribution/Entitlement)
  - Radius (Self/Others)
